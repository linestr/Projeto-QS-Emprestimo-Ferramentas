# Sistema de Empréstimo de Ferramentas – Análise de Qualidade de Software

## Descrição
Este repositório foi criado para a UC de Gestão e Qualidade de Software e tem como objetivo realizar a análise e validação da qualidade de um sistema legado desenvolvido na UC Programação de Soluções Computacionais.

## Objetivo
Aplicar técnicas e práticas de qualidade de software por meio do planejamento, execução e documentação de testes, visando identificar falhas, validar funcionalidades e avaliar a qualidade geral do sistema.

## Sistema Legado
Projeto original utilizado como base para os testes e análises:
https://github.com/andffc/Emprestimo-de-Ferramentas
